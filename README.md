🏥 Clínica Riviera Gomes - Projeto 01 (Landing Page)
Este projeto faz parte da disciplina de Desenvolvimento Web em Soluções para Clínica de Saúde. O objetivo foi criar uma Landing Page moderna, funcional e responsiva para uma clínica médica fictícia, aplicando conceitos fundamentais de Web Design e Lógica de Programação.


🚀 Tecnologias Utilizadas
O projeto foi construído utilizando as tecnologias fundamentais da web (Front-End):

HTML5: Estruturação semântica de todo o conteúdo.

CSS3: Estilização visual, utilizando Flexbox e CSS Grid para layout responsivo.

JavaScript (Vanilla): Lógica de captura de dados, persistência local e manipulação do DOM.


📋 Funcionalidades do Projeto
Apresentação Institucional: Descrição detalhada sobre os valores e a infraestrutura da Riviera Gomes.

Catálogo de Serviços: Exibição dinâmica de serviços como exames e consultas.

Equipe Médica: Galeria com fotos e especialidades dos profissionais (mínimo de 4 membros).

Agendamento de Consultas:

Formulário de captura (Nome, E-mail, Cidade, Estado).

Validação de campos obrigatórios via JS.

Persistência de Dados: Uso de localStorage para manter as informações do cliente durante a navegação.

Página de Confirmação: Redirecionamento dinâmico que exibe uma mensagem personalizada com o nome e e-mail capturados do formulário.


🛠️ Como rodar o projeto
Clone este repositório ou baixe os arquivos.

Certifique-se de que a estrutura de pastas está correta:

/html (Contém index.html e concluido.html)

/assets (Contém as imagens dos médicos e ícones)

style.css e app.js (Na raiz ou pasta indicada)

Abra o arquivo index.html em seu navegador de preferência ou utilize a extensão Live Server do VS Code.


🧠 Conceitos de ADS Aplicados
Nesta primeira entrega, foram exercitados conceitos importantes para a formação em sistemas:

Manipulação do DOM: Interação direta do JavaScript com elementos HTML.

UX (User Experience): Criação de caminhos intuitivos e feedback visual após ações do usuário.

Persistência Client-side: Armazenamento temporário de dados no navegador sem a necessidade de um banco de dados externo (ainda).
